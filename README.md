# Лабораторная работа 2, Задание 4

## Overview
Программа для обработки данных абитуриентов. Читает данные из текстового файла,
отбирает абитуриентов с проходным баллом больше 225 и сохраняет результаты в новый файл.
Реализована по модели КИС-2 (Клиент-Интерфейс-Сервер).

## Author
Иванов Иван, группа 11

## Usage
- Сборка программы: `make`
- Запуск программы: `./abiturient_program`
- Очистка: `make clean`

## Additional Notes
- Входной файл: `data/abiturients.txt`
- Выходной файл: `data/result.txt`
- Проект собирается с помощью make и GitHub Actions
